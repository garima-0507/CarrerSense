About CareerSense

CareerSense is an AI-based career recommendation system built using Machine Learning, Python, Flask, HTML, and CSS.

Users enter their skills, education, and interest domain. The ML model then processes this data and predicts the top 3 most suitable career paths.

Suggested roles include Machine Learning Engineer, Data Analyst, Web Developer, Cybersecurity Analyst, Android Developer, and Cloud Engineer.

The system uses Random Forest for classification and custom skill vectorization for better accuracy.
